## Connections

- Button: one side to `D7`, other side to `GND` (use internal pull-up in code or external resistor).
- Onboard LED: `D13` (no wiring needed).

## Diagram (ASCII)

```
 Arduino UNO R4
 +-----------+
 | D13 LED * |
 | D7  o-----+----[Button]----GND
 +-----------+
```

## Notes

- If not using internal pull-ups, add a 10k resistor to create a stable input.
